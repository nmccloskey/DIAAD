# POWERS Research Context

POWERS is a dialogue coding paradigm. DIAAD supports it as a human-reviewed workflow for organizing transcript-derived utterances, producing coding workbooks, summarizing completed coding, evaluating reliability, and calculating speaking-time-normalized rates.

## Human Coding And Automation

The module includes automated first-pass support for selected POWERS fields. This can reduce coding burden, especially for text-derived counts, but it should be treated as an expediter rather than a coder. Human review remains necessary because POWERS coding involves interpretive decisions and because automated NLP support can fail or misclassify language in ways that matter for analysis.

## Dialogue Context

POWERS is most relevant for clinician-client or other dialogic samples where interactional structure matters. Transcript tabularization gives the module utterance-level rows, speaker labels, and stable identifiers, but the research meaning of the coded values depends on the POWERS protocol and the study's conversational context.

## Reliability And Rates

Reliability evaluation compares completed primary and reliability coding. Analysis aggregates coding at several levels, and rates can normalize selected summary values by speaking time. These outputs help organize interpretation, but they do not replace decisions about what agreement is sufficient or which rate is meaningful for a research question.
